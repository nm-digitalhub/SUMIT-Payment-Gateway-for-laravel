HasCheckoutTheme

HasCheckoutTheme.php : 24

HasCheckoutTheme Trait

Provides dynamic checkout theme capabilities for Payable models. Allows customization of colors, branding, trust badges, and progress steps.

Usage:

class MayaNetEsimProduct extends Model implements Payable
{
    use HasCheckoutTheme;
}
