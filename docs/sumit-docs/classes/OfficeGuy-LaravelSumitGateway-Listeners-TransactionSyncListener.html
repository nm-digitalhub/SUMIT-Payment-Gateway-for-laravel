Transaction Sync Listener

Implements ADR-004: Handling Card Payments via SUMIT CRM Webhooks

Listens for SUMIT CRM Transaction webhooks and confirms card payments. This is the ONLY way to confirm card (token) payments, as SUMIT does not provide a dedicated card payment webhook like it does for Bit.

private findOrCreateTransaction(array<string|int, mixed> $payload, int|null $entityId) : OfficeGuyTransaction|null

This is the ONLY safe way to match SUMIT CRM Transaction cards to local OfficeGuyTransaction records. Matching by amount+timestamp is unsafe due to duplicate amounts, retries, recurring payments, etc.
